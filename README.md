# SMART SnippetFlow Landingpage

Eigenständiges Webprojekt für die Landingpage der Desktop-App SMART SnippetFlow.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- lucide-react für Icons

## Lokaler Start

```bash
npm install
npm run dev
```

Der lokale Server startet standardmäßig auf `http://127.0.0.1:5173`.

## Build

```bash
npm run build
```

## Repository vorbereiten

```bash
git init
git add .
git commit -m "Initial SMART SnippetFlow landingpage"
git branch -M main
git remote add origin <NEW_GITHUB_REPOSITORY_URL>
git push -u origin main
```
